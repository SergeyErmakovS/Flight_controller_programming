# Lua-скрипты для полётного контроллера

Раздел посвящён программированию полётного контроллера на языке Lua (ArduPilot Lua Scripting).

## Структура

```
lua/
├── README.md      — этот файл
├── examples/      — учебные примеры скриптов
├── scripts/       — рабочие скрипты для загрузки на контроллер
├── logs/          — логи выполнения скриптов
└── videos/        — видеоматериалы и демонстрации
```
