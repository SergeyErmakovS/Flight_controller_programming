# Flight_controller_programming

# Программирование БПЛА: Практика и обучение

Этот репозиторий содержит мои наработки, примеры кода и учебные проекты, созданные в рамках курса по программированию беспилотных летательных аппаратов.

## О чем этот проект
Здесь я исследую фундаментальные аспекты разработки ПО для дронов:
* **Полетные стеки:** Работа с PX4 и ArduPilot.
* **Связь:** Реализация MAVLink-протокола для телеметрии и команд.
* **Автономность:** Использование ROS (Robot Operating System) для задач планирования маршрута.
* **Стабилизация:** Эксперименты с ПИД-регуляторами и фильтрами Калмана (EKF).

## Стек технологий
* **Языки:** C++, Python
* **Инструменты:** ROS, MAVSDK, Gazebo (симуляция)
* **Аппаратная часть:** Полетные контроллеры (Pixhawk), одноплатные компьютеры (Raspberry Pi/Jetson)

## 📈 Цели обучения
Моя цель — разобраться в том, как превратить "железку" в интеллектуальную автономную систему, способную решать задачи компьютерного зрения и навигации.
